## Файлова структура

```

peleti.html       – Каталог продукти (пелети, екобрикети, въглища)
dostavka.html     – Информация за доставка и транспорт
kontakti.html     – Контактна форма и данни
za-nas.html       – За нас, предимства, числа
progress.md       – Прогрес по задачи (този файл)
```

## Технологии
- HTML5 (семантичен код)
- Tailwind CSS (CDN)
- Google Fonts: Inter (400–800)
- Цветова гама: оранжево (#E8620A), тъмно сиво (#1C1C1C), кафяво (#6B3E26), кремаво (#FDF6EE)

## Бележки
- Multi-page архитектура (5 HTML файла с единен header/footer)
- Placeholder изображения с описателни `alt` тагове
- Коментари в кода за всяка секция (улеснява прехвърляне в WordPress)
- Консистентна навигация със „За нас" линк добавен навсякъде
- Мобилно меню с анимиращ се хамбургер икона
